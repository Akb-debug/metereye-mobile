# MeterEye AI – Mobile App (Flutter)

This mobile application allows users to manage smart meters, configure ESP32 devices, and track electricity consumption in real time.

## 🚀 Features
- User authentication
- Meter creation and configuration
- Reading modes:
  - Manual
  - ESP32-CAM
  - Sensor
- QR code scanning for device onboarding
- ESP32 module association
- Device status monitoring
- Manual reading submission
- Consumption history

## 🛠️ Tech Stack
- Flutter
- Dart
- REST API (Spring Boot backend)

## 📱 Use Cases
- Register and configure a new meter
- Scan ESP32-CAM QR code
- Configure device workflow
- Monitor readings and consumption
